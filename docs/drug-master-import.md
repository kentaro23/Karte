# 医薬品マスタ — 現状と全保険収載薬の取込

## 現状（seed）
- 即利用可能な頻用薬 **150品目 / 140成分**（curated, 実名称・規格・剤形）
- 安全データ（禁忌・相互作用・極量）は **provenance厳格**：出典確認済みの主要成分のみ。
  未整備薬は処方時に黙ってPASSせず **「要確認 WARNING」**（＝安全側の設計通り）。
- 適応（効能・効果）：主要薬の標準的 ICD10 適応 **約100件** をシード
  （病名→適応薬リコメンドの根拠。PMDA添付文書 構造化取込で全置換）。

## 全保険収載医薬品（約2万品目）の取込

公的マスタは無料だが**利用登録が必要**で匿名の自動取得は不可（当環境で取得不能を確認）。
利用登録の上でファイルを入手し、1コマンドで全件投入できる（請求対応のレセ電コード/薬価/名称/区分）。

**入手元（いずれか）**
- 診療報酬情報提供サービス <https://shinryohoshu.mhlw.go.jp/> → 基本マスター → 医薬品マスター(y)
- 社会保険診療報酬支払基金「基本マスター」
- 厚労省「使用薬剤の薬価（薬価基準）収載品目リスト」（代替）

**取込（zip も csv も可・Shift_JIS自動変換）**
```
pnpm import:drugs /path/to/医薬品マスター.zip 2026-04
# または
pnpm import:drugs /path/to/y.csv
```
`importReceiptDrugMaster()` が `provenance=MHLW_RECEIPT` で全件 upsert（冪等）。
**安全データはこの経路では一切付与しない**（コード/名称/薬価/区分のみ）。禁忌・相互
作用・極量・適応は引き続き出典厳格（PMDA添付文書 等）でのみ拡充する。

## 取込後の挙動
- 全保険薬が処方ピッカーに出現（薬名検索）。
- 病名(ICD10)に適応のある薬は「適応薬（推奨）」として上位＋背景ハイライト表示
  （適応データが整備された範囲。未整備薬は通常表示・検索可）。
- 安全チェック（禁忌/相互/重複/極量/アレルギー）は出典済みデータのある薬で発火、
  無い薬は「要確認」提示（安全側）。
